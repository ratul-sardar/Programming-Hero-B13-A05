# Programming-Hero-B13-A05

Questions:

1️⃣ What is the difference between var, let, and const?
2️⃣ What is the spread operator (...)?
3️⃣ What is the difference between map(), filter(), and forEach()?
4️⃣ What is an arrow function?
5️⃣ What are template literals?


Answer:

1️⃣The main difference between var, let, and const is that: var can be re-declared, hoisted, and function-scoped. While let and const is block scoped, let allows the value to be reassigned, while const values can't be changed.
2️⃣spread operator will spread the value of an array or object into indivisual value or properties.
3️⃣map() will transform each value and return a new array without changing the old one, forEach() will perform an action to each element without returning anything and will change the original array.
4️⃣Arrow function is shortcut fucntion to the normal function that was added in js in the es6 update.
5️⃣Template literals a es6 feature that allows to write multiline string and also to put variables in the strings.
